RequestDirective

RequestDirective traverses a `$request` directive and turns the definitions there into executable functions.

Example

Example schema

{
 "$request":
   "nullRequest": {
     "href": "/dev/null",
     "method": "GET",
     "@onSuccess": "self.doSomething",
     "@onError": "self.fixError",
     "@params": "self.query"
   }
 }
